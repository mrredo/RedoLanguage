# RedoLanguage
Interpreter for redo langugae

# Features
### Methods:

- print(args...)
- println(args...)
- printf(format string, args...)
- free(variableKey string)

### Variable declaration
```ts
var key = value
var key = false
var key = "hello"
var key = 10
```
### Comments
supports
one line comments //

multi line comments

/*

*/

value can be a string, boolean or a number
